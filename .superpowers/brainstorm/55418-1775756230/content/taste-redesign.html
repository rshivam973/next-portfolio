Key Technical Decisions

  • Remove react-slick carousel → static grid with scroll animations
  • Remove FontAwesome → Phosphor Icons (light weight, consistent stroke)
  • Remove floating social sidebar → footer social links with proper aria-labels
  • Font swap: Raleway/News Cycle → Geist + Geist Mono (via next/font)
  • Color: Purple (#451952) → OLED black (#0e1011) + emerald accent
  • Keep: Framer Motion (already installed), EmailJS, react-scroll, Tailwind
  • Add: @phosphor-icons/react, Geist font
